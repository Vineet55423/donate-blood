================================================================================
                  FINDBLOOD - BLOOD DONATION PLATFORM
                         QUICK START GUIDE
================================================================================

📦 Thank you for downloading FindBlood!

This is a complete React-based blood donation platform ready to run on your
local computer.

================================================================================
                        🚀 GET STARTED IN 3 STEPS
================================================================================

STEP 1: Install Node.js (if you haven't already)
────────────────────────────────────────────────
→ Visit: https://nodejs.org/
→ Download the LTS version
→ Install and restart your computer

Verify installation:
  Open terminal and type: node --version
  You should see: v18.x.x or higher


STEP 2: Install Project Dependencies
────────────────────────────────────────────────
→ Open terminal/command prompt
→ Navigate to this folder:
  
  cd path/to/find-blood-platform

→ Install packages:
  
  npm install

  (This takes 2-5 minutes)


STEP 3: Start the Application
────────────────────────────────────────────────
→ In the same terminal, run:
  
  npm run dev

→ Open your browser and go to:
  
  http://localhost:5173/

🎉 That's it! The app should now be running!

================================================================================
                        🔧 TROUBLESHOOTING
================================================================================

❌ "npm: command not found"
   → Node.js not installed. Install from nodejs.org

❌ Port already in use
   → Vite will use next available port automatically

❌ Blank white page
   → Press F12, check console for errors
   → Try: npm install again

❌ Slow installation
   → Normal! Installing 300+ packages takes time
